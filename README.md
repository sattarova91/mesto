# Проект 4: Место

### Описание проекта:
  Mesto - интерактивная страница с возможностью редактировать профиль. В дальнейшем можно будет ставить "лайки", добавлять новые фотографии.
### Использованные технологии:
+ Методология БЭМ;
+ Адаптивная верстка;
+ ReactJS
